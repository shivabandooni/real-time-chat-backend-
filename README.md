# Real-Time Chat System (FastAPI)

## 📌 Description
A real-time chat application built using FastAPI and WebSockets. It supports multiple users and private messaging with instant communication.

---

## 🚀 Features
- Real-time messaging using WebSocket
- Multi-user support
- Private user-to-user chat
- Simple frontend using HTML & JavaScript

---

## 🛠️ Tech Stack
- Python (FastAPI)
- WebSocket
- HTML, JavaScript

---

## ▶️ How to Run

1. Install dependencies:
pip install fastapi uvicorn

2. Run server:
uvicorn main:app --reload

3. Open index.html in browser

---

## 💡 How to Use
- Enter your username
- Connect to server
- Send messages to another user

---

## 📷 Demo
Open two browser windows and chat between users in real-time.
